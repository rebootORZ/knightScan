# knightScan
# 应用场景:  
当进入内网后，无法通过常规手段（弱口令、PTH、系统漏洞）横向时，需要打点，那么就需要先进行资产探测。  
# 解决问题：  
一般常见的工具都是探测端口，需要自己再去访问才知道具体系统。写这个工具的目的就是直接内置常见web端口，根据给定IP范围（CIDR）探测端口开放情况，再查看web服务的title,从而帮助攻击队友快速定位重要系统。  
# 注：  
探测端口采用ICMP，可能需要管理员权限。  
  
# Usage:  
go run main.go -h 192.168.1.0/24  
go run main.go -h 192.168.1.0/16
